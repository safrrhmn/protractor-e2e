Welcome to Protractor-e2e

As Angular is becoming one of the key technologies in web development, it is important for an Automation Testser to adopt the strategies for testing this. This repo follow simply the Page Object pattern along with Jasmine and protractor for testing Angular site. Feel free to fork me.
